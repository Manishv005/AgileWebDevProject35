Welcome to PuzzleMe.

"Rule's to play this game are."

Rule 1:

Guess the word by selecting letters one at a time.

Rule 2:

Each correctly guessed letter will replace a corresponding dash(-) in the word.

Rule 3:

Each incorrect guess will contribute to the drawing of the hangman.

Rule 4:

The game ends in success if you correctly guess all letters of the word before the hangman is fully drawn.

Rule 5:

The game ends in failure if the hangman is fully drawn before the word is guessed.

Rule 6:

The timer starts at the beginning of each game and stops when the game ends. The time taken affects your score and ranking on the leaderboard.
